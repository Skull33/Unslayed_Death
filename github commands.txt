para hacer un nuevo repositorio
----------------------------------------------------
echo "# Dnur" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Dnursaurio/Dnur.git
git push -u origin main
----------------------------------------------------

----------------------------------------------------
git remote add origin https://github.com/Dnursaurio/Dnur.git
git branch -M main
